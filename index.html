Overview

Artscape Youngplace is a non-profit organization that nurtures creation, learning, and collaboration through innovative programs, experiences, and events in an inspiring, social environment to strengthen and grow their community. However, the current website lacks easy navigation for users.

This project aimed to redesign the Artscape Youngplace website to improve user experience and navigation. The goal was to create a user-friendly interface that provides easy access to events, exhibitions, and information.

The project involved extensive user research, UX design, UI design, and iterative prototyping. By implementing a well-structured information architecture and intuitive navigation, the redesigned website successfully reduced the time users spend searching for information.

The redesigned wireframes and interactive prototypes showcase the enhanced user flow and improved visual design, contributing to a more engaging experience for Artscape Youngplace visitors.

Context

Problem Statement

The navigation system's flaws in categorization and the excessive information overwhelm users, leading to frustration and prolonged search times for the desired information.

User Research

The User wants to search for the upcoming events happening in toronto for herself as an Artist. She searches the ArtScape Youngplace website through Google. After opening the Artscape Youngplace site, she clicks What's On, and Event Calendar then she comes to the new page. After clicking the Exhibitions on this page, She arrives at the page showing the events Sections. After clicking Event picture, she reaches to the event details page..The User sees unorganized details of the events on the Artscape Youngplace website.
